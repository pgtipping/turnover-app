Guide to Using the Employee Turnover Calculator

Welcome! The employee turnover calculator allows you to calculate and analyze employee turnover rates over a selected period, using monthly data on voluntary exits and end-of-month headcounts.

This guide explains how to use the calculator.

Before using the app, ensure you have the starting headcount for the first month and monthly data for total exits and month-end headcounts.

How to Use the Calculator

Manual Data Entry

  1. Enter the initial headcount for the start month.
  2. Choose a data entry method: manual or CSV upload.
  3. Select the start and end months for your analysis period.
  4. Click "Generate Form" to display a form with columns for Month, Beginning (prepopulated), Voluntary Exits, and End.
  5. Enter the number of exits and the ending headcount for each month. The beginning headcount for the next month updates automatically.

CSV File Upload

  1. Enter the initial headcount for the start month.
  2. Choose "CSV Upload" as the data entry method.
  3. Prepare a CSV file with the following columns, with no headers: Month (as a number from 1 to 12), Voluntary Exits, and End Headcount.
  4. Click "Choose File" to upload your CSV.
  5. Click "Upload File" to import the data into the form.

Other Steps: Manual Entry and CSV Upload

  1. Select your chart options: monthly rates, quarterly rates, YTD rates, or annualized rates.
  2. Click "Submit" to view the calculated rates, a chart, and the option to generate a PDF.
  3. Click "Generate PDF" to download the entire page with the turnover data table, calculated rates, and chart visualization.
